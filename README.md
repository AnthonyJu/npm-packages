# npm packages

## eslint-config

### 安装

```bash
pnpm install --save-dev @anthony-ju/eslint-config
```

### 使用

创建`.eslintrc`文件，并将以下内容添加到文件中:

```json
{
  "extends": "@anthony-ju/eslint-config"
}
```
### 感谢巨人

此配置继承至自 [@antfu/eslint-config](https://github.com/antfu/eslin-config) ❤️
